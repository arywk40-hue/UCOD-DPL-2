\begin{abstract}
%==============================================================================
Unsupervised Camouflaged Object Detection (UCOD) aims to segment hidden targets without human annotations. Existing self-supervised frameworks rely heavily on fixed pseudo-labeling heuristics and scalar weighting functions that assume spatially uniform label noise. However, pseudo-label noise in camouflaged scenes is inherently non-stationary, concentrating around complex boundaries, low-contrast textures, and small instances.

In this research exploration, we propose \textbf{\methodname{} (Agentic Decision Framework)}, an experimental architecture investigating whether an autonomous multi-agent decision system can adaptively learn spatial supervision maps $W_i(x,y) \in [0,1]^{H \times W}$. Our system incorporates four specialized training agents: (1) a \textbf{Quality Assessment Agent (QAA)} utilizing multi-scale ASPP convolutions and feature gradient sensing; (2) a \textbf{Region Analysis Agent (RAA)} analyzing local background context ring contrast; (3) a \textbf{Memory Agent (MA)} tracking dual-temporal mean and variance dynamics per training sample; and (4) an \textbf{Attention-Gated Decision Agent (DA)} with CBAM attention gating. We formalize the complete mathematical design, loss functions ($\mathcal{L}_{conf}$, $\mathcal{L}_{region}$, $\mathcal{L}_{temporal}$), and an empirical benchmark evaluation protocol. This working paper documents our architectural formulation prior to full GPU cluster training and validation.
\end{abstract}

%==============================================================================
\section{Research Motivation \& Questions}
%==============================================================================

Camouflaged Object Detection (COD) requires segmenting targets visually seamlessly integrated into their surroundings~\cite{sinet,zoomnet}. Unsupervised learning eliminates reliance on expensive pixel-level labels~\cite{camoteacher}, but introduces severe challenges regarding pseudo-label reliability.

Traditional approaches apply global scalar mixing weights across the spatial grid. We hypothesize that scalar mixing is sub-optimal for camouflaged objects because pseudo-label quality varies drastically across image regions. 

This research exploration is guided by three primary Research Questions (RQs):
\begin{itemize}
    \item \textbf{RQ1 (Spatial vs. Global Calibration):} Does replacing global scalar weighting with dense spatial supervision maps $W_i(x,y)$ improve segmentation boundary precision and target recall?
    \item \textbf{RQ2 (Multi-Agent Synergies):} How do quality assessment (QAA), background context contrast (RAA), and temporal memory (MA) interact when fused via attention gating (DA)?
    \item \textbf{RQ3 (Computational Efficiency):} Can multi-agent supervision reasoning be conducted entirely during training to preserve zero inference overhead?
\end{itemize}

\subsection{Agent 1: Quality Assessment Agent (QAA)}

QAA predicts dense spatial reliability $R_i \in [0,1]^{H \times W}$ using ASPP convolutions across dilation rates $\{1, 2, 4, 6\}$:
\begin{equation}
R_i = \text{QAA}\left([\hat{P}_i^{fs}, \sigma(\hat{P}_i^t), \sigma(\hat{Y}_i^{FG}), D_i, H_i, \|\nabla F_i\|]\right)
\end{equation}
where $D_i = |\sigma(\hat{P}_i^t) - \hat{P}_i^{fs}|$, entropy $H_i = -p\log p - (1-p)\log(1-p)$, and $\|\nabla F_i\|$ is spatial feature gradient magnitude sensing local visual ambiguity. Self-supervised calibration is enforced via:
\begin{equation}
\mathcal{L}_{conf} = \text{BCE}\left(R_i(x,y),\ \mathbf{1}\left[D_i(x,y) < 0.3\right]\right)
\end{equation}

\subsection{Agent 2: Region Analysis Agent (RAA)}

RAA evaluates region topology and background ring contrast for connected components $c_k^{FG}$:
\begin{equation}
\text{contrast}_k = \|\bar{F}_{c_k} - \bar{F}_{\text{bg\_ring}}\|_2
\end{equation}
where $\text{bg\_ring}$ is a 5-pixel dilated background ring around $c_k$. Regions are classified into Easy, Hard, and Ambiguous categories, penalizing prediction variance on confident regions via $\mathcal{L}_{region}$.

\subsection{Agent 3: Memory Agent (MA)}

MA tracks CPU-side sample-level prediction mean $M_i^{(t)}$ and running variance $V_i^{(t)}$ across training epochs:
\begin{align}
M_i^{(t)} &= \alpha_m M_i^{(t-1)} + (1-\alpha_m) \sigma(\hat{Y}_i^{FG,(t)}) \\
V_i^{(t)} &= \alpha_m V_i^{(t-1)} + (1-\alpha_m) \left(\sigma(\hat{Y}_i^{FG,(t)}) - M_i^{(t-1)}\right)^2
\end{align}
yielding stability $S_i^{temp} = 1 - |\sigma(\hat{Y}_i^{FG}) - M_i^{(t-1)}|$ and variance $\sigma^2_{temp} = V_i^{(t)}$ with loss $\mathcal{L}_{temporal}$.

\subsection{Agent 4: Decision Agent (DA)}

DA applies CBAM spatial and channel attention over fused evidence $[R_i, \text{region\_map}, S_i^{temp}, \sigma^2_{temp}, t/T]$ to produce spatial mixing maps $W_i(x,y)$:
\begin{equation}
P_i(x,y) = W_i(x,y) \cdot \sigma(\hat{P}_i^t) + (1 - W_i(x,y)) \cdot \hat{P}_i^{fs}
\end{equation}

\subsection{Total Loss Formulation}
\begin{equation}
\mathcal{L}_{total} = \mathcal{L}_{seg} + \mathcal{L}_\bot + 0.5 \mathcal{L}_{conf} + 0.1 \mathcal{L}_{region} + 0.3 \mathcal{L}_{temporal}
\end{equation}

\section{Conclusion \& Next Steps}
%==============================================================================

This research exploration presents the mathematical architecture and multi-agent formulation of \methodname{}. By replacing rigid scalar supervision with four collaborating training agents (QAA, RAA, MA, DA), our goal is to investigate spatial pseudo-label calibration in unsupervised camouflaged object detection. The next phase will execute GPU cluster training to evaluate these research hypotheses across standard COD benchmarks.
